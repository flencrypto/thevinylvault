OVERVIEW
══════════════════════════════════════════════════════════════════════

VinylVault is a fully client-side app (Vite + React SPA). There is no
backend server. All API credentials are stored in your browser's
localStorage, or in GitHub Spark KV (for server-side Spark features).

HOW TO CONFIGURE:
  → Open the app → click the "Settings" tab in the bottom navigation
  → Scroll to the relevant integration section
  → Paste your key/token and click Save

IMPORTANT: Never commit secrets to version control. All keys listed
below are stored ONLY in the user's browser (localStorage) or Spark KV.

[4] DISCOGS PERSONAL ACCESS TOKEN  ★ REQUIRED for marketplace features
══════════════════════════════════════════════════════════════════════

Feature(s) unlocked:
  • Record valuation (estimated values from Discogs marketplace)
  • Pressing identification via Discogs database
  • Marketplace search and price history
  • Collection import from Discogs
  • Bargain scanning (Discogs marketplace)
  • DiscogsTestDialog, DiscogsMarketAnalysisDialog
  • WatchlistView Discogs searches
  • BargainsView Discogs deal scanning

Required credential:
  localStorage key : discogs_personal_token
  (Alternative OAuth keys below)

OAuth required: Optional — Personal Access Token is simpler

Where to get it (Personal Token — recommended):
  1. Log in at https://www.discogs.com/
  2. Go to https://www.discogs.com/settings/developers
  3. Scroll to "Personal Access Tokens" section
  4. Click "Generate new token"
  5. Copy the token
  6. Open VinylVault → Settings → Discogs Integration → paste token → Save

Where to get it (OAuth — advanced):
  1. Go to https://www.discogs.com/settings/developers
  2. Register a new application to get Consumer Key + Consumer Secret
  3. Open VinylVault → Settings → Discogs → enter Consumer Key + Secret

OAuth credentials:
  localStorage key : discogs_consumer_key
  localStorage key : discogs_consumer_secret

Key format:   Long alphanumeric string
Rate limits:  60 requests/minute (authenticated), 25/minute (unauthenticated)
Free account: Yes, free Discogs account is sufficient

Which UI actions depend on this:
  • "Get Value" / valuation on any record
  • "Scan for Bargains" in BargainsView
  • "Search Discogs" in WatchlistView
  • Pressing identification searches
  • Market analysis dialogs

Notes:
  - Without this token, Discogs calls will fail or be heavily rate-limited.
  - The DiscogsSetupGuide component in the app guides users through setup.

Where to get it:
  1. Go to https://developer.ebay.com/
  2. Sign in with your eBay account (or create one)
  3. Navigate to My Account → Application Keys
  4. Click "Create an App ID"
  5. Select "Production" environment
  6. Copy the "App ID (Client ID)"
  7. Open VinylVault → Settings → eBay Integration → paste App ID → Save

Key format:   YourName-AppName-PRD-xxxxxxxxxxxxxxx-xxxxxxxx
Cost:         Free Developer Program account
Docs:         https://developer.ebay.com/develop/apis/restful-apis/buy-apis

Which UI actions depend on this:
  • "Scan eBay" in DealScannerView / BargainsView
  • eBay price comparison in MarketplaceComparisonView
  • eBay bargain cards in BargainCard component

Notes:
  - Uses the eBay Finding API (browse/search), not the Trading API
  - No OAuth flow needed for read-only search
  - Rate limits: 5,000 calls/day on free tier

  • Viewing and managing minted NFTs in NFTView
  • Blockchain-verified eBay listings

Required setup:
  A Solana browser wallet extension installed and connected:
    • Phantom Wallet:   https://phantom.app/         (window.solana)
    • Solflare:         https://solflare.com/         (window.solflare)
    • Backpack:         https://backpack.app/         (window.backpack)

OAuth / Login required: Yes — wallet connect (browser extension)

How to connect:
  1. Install a Solana wallet browser extension (Phantom recommended)
  2. Create or import a wallet
  3. Fund with SOL for transaction fees (~0.01 SOL per mint)
  4. Open VinylVault → NFT View → "Connect Wallet"
  5. Approve the connection in the wallet extension

Notes:
  - Mainnet-beta is the default network for production mints
  - Each NFT mint costs ~0.01–0.02 SOL in network fees
  - The WalletConnect component handles wallet detection

MISSING KEY UX BEHAVIOR
══════════════════════════════════════════════════════════════════════

When a user triggers a feature that requires a missing credential:

1. The app checks for the key in localStorage / Spark KV
2. If missing, an IntegrationGateDialog is shown with:
   - Which key is missing
   - Why it's needed
   - Direct link to Settings → relevant section
3. No silent failures — every gated action explains what's required
4. The Settings tab provides fields for all credential inputs
